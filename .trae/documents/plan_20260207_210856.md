# Plano de Implementação: IZU Brindes E-commerce

Este plano detalha a construção do site institucional e catálogo de produtos para a HM Brindes, utilizando React, TypeScript, Tailwind CSS e Supabase.

## 1. Configuração Inicial do Projeto

* **Inicialização**: Criar projeto Vite com React e TypeScript.

* **Dependências**: Instalar e configurar:

  * `tailwindcss`, `postcss`, `autoprefixer` (Estilização)

  * `shadcn/ui` (Componentes de UI)

  * `react-router-dom` (Roteamento)

  * `zustand` (Gerenciamento de Estado)

  * `@supabase/supabase-js` (Banco de Dados e Auth)

  * `lucide-react` (Ícones)

  * `framer-motion` (Animações)

  * `react-hook-form` + `zod` (Formulários e validação)

* **Variáveis de Ambiente**: Configurar `.env` com credenciais do Supabase e WhatsApp.

## 2. Banco de Dados e Backend (Supabase)

* **Schema**: Criar script SQL para a tabela `products` com todos os campos especificados (dimensões, cores, imagens, etc.).

* **Storage**: Configurar bucket `products` para imagens com políticas de acesso (leitura pública, escrita autenticada).

* **Segurança**: Configurar Row Level Security (RLS) para proteger a escrita de dados.

## 3. Arquitetura e Componentes Base

* **Layout**: Header (com navegação e dropdowns de categorias), Footer (contato e links).

* **UI Kit**: Implementar componentes base (Button, Input, Card, Dialog, Select, etc.) via shadcn/ui.

* **Hooks**: Criar hooks customizados para acesso ao Supabase (`useProducts`, `useAuth`).

## 4. Implementação das Páginas Públicas

* **Homepage**:

  * Hero section com CTA.

  * Grid de categorias principais.

  * Seções institucionais (Parceiros, História, Depoimentos).

* **Catálogo de Produtos (`/produtos`)**:

  * Listagem com filtros por Categoria/Subcategoria.

  * Busca de produtos.

  * Cards de produto com imagem e nome.

* **Detalhes do Produto (`/produto/:id`)**:

  * Galeria de imagens interativa.

  * Seletor de cores e visualização de especificações.

  * **Botão WhatsApp**: Lógica para gerar link com mensagem pré-preenchida com o nome do produto e demais informações que faça sentido na mesnagem do WhatsApp.

* **Páginas Institucionais**: FAQ (Accordion) e Sobre.

## 5. Painel Administrativo (`/admin`)

* **Autenticação**: Página de Login integrada ao Supabase Auth.

* **Proteção de Rotas**: Componente `ProtectedRoute` para áreas restritas.

* **Gerenciamento de Produtos**:

  * Listagem com tabela e ações (Editar/Excluir/Ativar/Desativar).

  * **Formulário de Produto**:

    * Upload de múltiplas imagens (Supabase Storage).

    * Seleção de Cores e Categorias.

    * Campos técnicos (dimensões, peso).

## 6. Polimento e Entrega

* **SEO**: Configurar meta tags básicas.

* **Responsividade**: Ajustes mobile-first.

* **Documentação**: Criar README e instruções de uso.
